# Тестовое задание Mindbox - Todo List

## Функциональность

- ✅ Создание новых задач
- ✅ Редактирование существующих задач
- ✅ Удаление задач
- ✅ Перемещение задач вверх/вниз в списке
- ✅ Фильтрация задач (все/активные/завершенные)
- ✅ Очистка завершенных задач
- ✅ Сохранение задач в localStorage
- ✅ Адаптивный дизайн
- ✅ Анимации с использованием Framer Motion
- ✅ Unit-тесты с использованием Jest и React Testing Library

## Технологии

- React 18
- TypeScript
- Emotion (styled-components)
- Framer Motion
- Jest
- React Testing Library

## Установка и запуск

1. Установите зависимости:

```bash
npm install
```

2. Запустите приложение:

```bash
npm start
```

Приложение будет доступно по адресу: http://localhost:5173

## Тестирование

Для запуска тестов выполните:

```bash
npm test
```

## Структура проекта

```
src/
  ├── components/          # React компоненты
  │   ├── __tests__/      # Тесты компонентов
  │   ├── TodoList.tsx    # Основной компонент списка задач
  │   ├── TodoItem.tsx    # Компонент отдельной задачи
  │   ├── TodoForm.tsx    # Форма добавления задачи
  │   ├── TodoFilters.tsx # Фильтры задач
  │   └── TodoHeader.tsx  # Заголовок приложения
  ├── types/              # TypeScript типы
  │   └── todo.ts         # Типы для задач
  └── App.tsx             # Корневой компонент
```

## Требования

- Node.js 18 или выше
- npm 9 или выше
